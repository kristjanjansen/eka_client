## EKA Classifier

A messaging client written in VueJS.

Part on the [Estonian Academy of Arts](https://www.artun.ee/en/home/) Emerging Themes masters project.

### Demo

https://eka-client.now.sh

### Installation and running locally

```sh
npm i
npm run start
```

### Deploying

```sh
npm i -g now
now
now alias
```
